## 🧳 ТУРИСТ USER JOURNEY

### Қадам 1: Landing Page → Таңдау
```
[Landing] ──("Турист" басу)──▶ [Авторизация]
```
**Орындалатын әрекет**: Пайдаланушы Landing бетіндегі "Турист ретінде бастау" түймесін басады.

### Қадам 2: Авторизация → Карта
```
[Авторизация] ──("Кіру"/"Тіркелу")──▶ [Интерактивті карта]
```
**Орындалатын әрекет**: Email/құпиясөз немесе Google OAuth арқылы авторизациядан өту.

### Қадам 3: Карта → Маршрут таңдау
```
[Карта] ──("Маршрут" басу)──▶ [Маршрут сипаттамасы]
```
**Орындалатын әрекет**: Картадағы маршрутты немесе sidebar-дағы карточканы таңдау.

### Қадам 4: Маршрут → Брондау/Жүру
```
[Маршрут сипаттамасы] ──("Гидпен жүру")──▶ [Брондау]
[Маршрут сипаттамасы] ──("Өз бетінше")──▶ [Навигация бастау]
```
**Орындалатын әрекет**: Екі нұсқадан біреуін таңдау — гидпен немесе өздігінен.

### Қадам 5: Брондау → Кабинет
```
[Брондау] ──("Төлем жасау")──▶ [Жеке кабинет]
```
**Орындалатын әрекет**: Гид, күн, уақыт таңдап, төлемді жасау.

---

## 🎯 ГИД USER JOURNEY

### Қадам 1: Landing → Тіркелу
```
[Landing] ──("Гид болу")──▶ [Гид тіркелу формасы]
```
**Орындалатын әрекет**: "Гид болғым келеді" түймесін басып, тіркелу формасын толтыру.

### Қадам 2: Тіркелу → Верификация
```
[Тіркелу] ──("Құжат жүктеу")──▶ [Верификация күту]
```
**Орындалатын әрекет**: Жеке куәлік, гид сертификаты жүктеу.

### Қадам 3: Верификация → Dashboard
```
[Верификация OK] ──▶ [Гид Dashboard]
```
**Орындалатын әрекет**: Админ құжаттарды тексергеннен кейін гид статусы белсенді болады.

### Қадам 4: Dashboard → Тур құру
```
[Dashboard] ──("Жаңа тур")──▶ [Тур құру формасы]
```
**Орындалатын әрекет**: Жаңа тур атауы, сипаттамасы, бағасын енгізу.

### Қадам 5: Тур → Күнтізбе
```
[Тур құру] ──("Сақтау")──▶ [Күнтізбе басқару]
```
**Орындалатын әрекет**: Қолжетімді күндерді белгілеу, брондар қабылдау.

## 🎨 UI КОМПОНЕНТТЕР ТІЗІМІ

### Navigation компоненттері
- `Header` — Негізгі навигация
