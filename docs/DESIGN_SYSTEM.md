## 🌿 ДИЗАЙН ФИЛОСОФИЯСЫ

### Негізгі принциптер

| Принцип | Сипаттама |
|---------|-----------|
| **Eco-friendly** | Табиғи түстер, органикалық формалар, "тыныс алатын" кеңістік |
| **Минимализм** | Тек қажетті элементтер, ақ кеңістік маңызды |
| **Accessibility** | Жоғары контраст, үлкен touch targets (min 44px) |
| **Native Feel** | iOS/Android дизайн гайдлайндарына сәйкес |

### Визуал метафоралары

```
🌲 Орман     → Тыныштық, табиғилық
🍃 Жапырақ   → Жаңару, эко-сана
🗺️ Карта     → Серігер, жол
💧 Су тамшы → Тазалық, ресурс
```

## 📝 ТИПОГРАФИЯ

### Font Family

```css
--font-primary: 'Inter', system-ui, sans-serif;
--font-mono: 'Fira Code', monospace;
```

### Type Scale

| Стиль | Өлшем | Weight | Line Height | Қолданылуы |
|-------|-------|--------|-------------|------------|
| **Display** | 32px | 800 | 1.2 | Hero headers |
| **H1** | 28px | 700 | 1.3 | Page titles |
| **H2** | 24px | 700 | 1.3 | Section titles |
| **H3** | 20px | 600 | 1.4 | Card titles |
| **H4** | 18px | 600 | 1.4 | Subsections |
| **Body Large** | 16px | 400 | 1.5 | Main content |
| **Body** | 14px | 400 | 1.5 | Default text |
| **Caption** | 12px | 500 | 1.4 | Labels, hints |
| **Overline** | 10px | 700 | 1.2 | Badges, tags |

### Mobile Specific

```css
/* iOS Dynamic Type support */
body {
  -webkit-text-size-adjust: 100%;
}

/* Minimum readable size */
.body-text {
  font-size: max(14px, 1rem);
}
```

## ✨ АНИМАЦИЯЛАР

### Transition Timing

| Түрі | Duration | Easing | Қолданылуы |
|------|----------|--------|------------|
| Micro | 150ms | ease-out | Hover, press states |
| Standard | 200ms | ease-in-out | Page elements |
| Emphasis | 300ms | cubic-bezier(0.4, 0, 0.2, 1) | Modals, bottom sheets |
| Complex | 400ms | spring(1, 100, 12) | Route animations |

### Key Animations

```css
/* Button press */
.btn:active {
  transform: scale(0.96);
  transition: transform 150ms ease-out;
}

/* Card hover (desktop) */
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 32px rgba(34, 197, 94, 0.2);
  border-color: #22C55E;
  transition: all 200ms ease-out;
}

/* Bottom sheet drag */
.bottom-sheet {
  transition: transform 300ms cubic-bezier(0.4, 0, 0.2, 1);
}

/* Route line draw */
.route-line {
  stroke-dasharray: 1000;
  stroke-dashoffset: 1000;
  animation: draw 2s ease-out forwards;
}

@keyframes draw {
  to { stroke-dashoffset: 0; }
}

/* POI marker bounce */
.poi-marker {
  animation: bounce 300ms ease-out;
}

@keyframes bounce {
  0% { transform: scale(0) translateY(10px); }
  60% { transform: scale(1.1) translateY(-5px); }
  100% { transform: scale(1) translateY(0); }
}

/* Skeleton loading */
.skeleton {
  background: linear-gradient(90deg, #ECFDF5 25%, #F0FDF4 50%, #ECFDF5 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```
