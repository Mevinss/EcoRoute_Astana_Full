## Интерфейс / UI Changes

### Жаяу Режим (Walking Mode) 🚶

**Мүмкіндіктер / Features:**
- ✅ Кез келген екі нүктеге маршрут құру
- ✅ OSRM динамикалық маршрут есептеу
- ✅ Картаны басып нүктелерді таңдау

**Пайдалану / Usage:**
1. "Жаяу" батырмасын таңдаңыз
2. Картада бастапқы нүкте басыңыз (🟢)
3. Картада соңғы нүкте басыңыз (🔴)
4. "Маршрутты есептеу" батырмасын басыңыз

---

### Велосипед Режимі (Cycling Mode) 🚴

**Мүмкіндіктер / Features:**
- ✅ **2 дайын маршрутты таңдау** (жаңа!)
- ✅ Немесе картадан өз маршрутыңызды құру

**Пайдалану / Usage:**

#### Опция 1: Дайын маршрут (Recommended)
1. "Велосипед" батырмасын таңдаңыз
2. Дайын маршруттар тізімінен біреуін таңдаңыз:
   - "Велосипед: Президенттік Парк — Ботаникалық Бақ"
   - "Велосипед: Байтерек — EXPO Алаңы"
3. Маршрут лезде картада көрінеді! ✨

#### Опция 2: Өз маршрутыңыз
1. "Велосипед" батырмасын таңдаңыз
2. Дайын маршруттарды өткізіп жіберіңіз
3. Картада бастапқы нүкте басыңыз
4. Картада соңғы нүкте басыңыз
5. "Маршрутты есептеу" батырмасын басыңыз

### 3. Frontend Map Component (`frontend/src/app/map/page.tsx`)
```typescript
// Added state for predefined route selection
const [selectedPredefinedRoute, setSelectedPredefinedRoute] = useState<any>(null);

// Added query for cycling routes
const { data: cyclingRoutesData } = useQuery({
  queryKey: ['cycling-routes'],
  queryFn: async () => {
    const res = await api.get('/routes?type=cycling');
    return res.data;
  },
  enabled: navMode === 'cycling' && plannerOpen,
});

// Added UI for cycling route selector
{navMode === 'cycling' && cyclingRoutesData?.routes && (
  <div>
    {/* Predefined routes list */}
    {cyclingRoutesData.routes
      .filter((r: any) => r.name_kz.includes('Велосипед:'))
      .map((route: any) => (
        <button onClick={() => handleSelectPredefinedRoute(route)}>
          {route.name_kz}
        </button>
      ))}
  </div>
)}
```

## Артықшылықтар / Benefits
